# Student API System

## Overview
This project is a simple API system built with Python and Flask.

## Skills
- Python
- Flask
- JSON
- Backend Development
- API Development

## Description
This project provides a simple student management API.
Users can retrieve student information and add new student data.

## Functions
- Get all students
- Get one student by ID
- Add a new student

## How to Run
1. Install Python
2. Install Flask:
   pip install flask
3. Run:
   python app.py

## API Endpoints
- GET /students
- GET /students/<id>
- POST /students
